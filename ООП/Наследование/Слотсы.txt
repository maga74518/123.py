__slots__ в объектно-ориентированном программировании (ООП) - это специальный атрибут класса в Python, который позволяет
 явно определить набор атрибутов (и их типов), которые могут быть установлены для экземпляров класса.

Определяя __slots__ в классе, вы можете уменьшить объем используемой памяти для экземпляров класса, так как Python
 больше не будет выделять память для словаря, который обычно используется для хранения атрибутов объекта.

Кроме того, определение __slots__ также может помочь обнаруживать ошибки и предотвращать динамическое добавление новых
 атрибутов к экземплярам класса, что может быть полезно в определенных случаях.

class MyClass:
    __slots__ = ['attr1', 'attr2']

    def __init__(self, val1, val2):
        self.attr1 = val1
        self.attr2 = val2

obj = MyClass(1, 2)
print(obj.attr1)  # 1
print(obj.attr2)  # 2

# Попытка добавить новый атрибут вызовет ошибку
obj.attr3 = 3  # AttributeError: 'MyClass' object has no attribute 'attr3'
